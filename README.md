# chemeng-final-project
Static Soft Sensor and Decision Support Models: Product Purity Deviation Detection and Reflux Ration Adjustment


## Project Overview
This project develops a two-hidden-layer neural network soft sensor for a debutanizer that predicts distillate purity deviation for a 99.5 mol% polymer-grade specification. In addition, it develops a complementary binary decision-support tool that recommends corrective reflux action using steady-state process variables.


## Project Objective
The ability to consistently produce a n-butane distillate purity of 99.5 mol % is complicated by feed disturbances that shift operating conditions while online gas chromatography analyzers report updated compositions up to 15 minutes after the fact. Thus, this project uses simulation-generated data and machine learning models to help estimate process behavior and support decision-making.

## Project Approach
A steady-state debutanizer column was modeled in DWSIM using a ternary hydrocarbon mixture. Data was generated by varying operating conditions through sensitivity studies. The resulting dataset was used to train machine learning models for prediction and classification tasks related to purity control.
